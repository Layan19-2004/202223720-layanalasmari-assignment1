# Technical Documentation

## Technologies Used
- **HTML5**: Builds the structure of the portfolio website (sections, sssnavigation, content).
- **CSS3**: Styles the website layout, typography, colors, and responsiveness.
  - Uses **Flexbox** for navigation layout.
  - Uses **media queries** for mobile responsiveness.
- **JavaScript (Vanilla JS)**: Adds interactivity to the website (frontend only).
- **VS Code**: Development environment used to write and manage project files.
- **Live Server (VS Code Extension)**: Used to run the website locally with automatic refresh.

## How to Run / Install / Operate the Web Application

### Requirements
- A web browser (Chrome, Edge, Firefox, or Safari)
- VS Code (recommended)
- Live Server extension (recommended)

### Option 1: Run using Live Server (Recommended)
1. Open the project folder in **VS Code**.
2. Install the **Live Server** extension if it is not installed.
3. Open `index.html`.
4. Right click inside `index.html` and select **Open with Live Server**.
5. The website will open in your browser and refresh automatically when you save changes.

### Option 2: Run by opening the HTML file directly
1. Locate `index.html` in your project folder.
2. Double-click `index.html` to open it in your browser.

> Note: Live Server is recommended because it reloads changes automatically.

### How to Use the Website
- Use the navigation bar to move between sections (About, Projects, Online Courses, Contact).
- Read the content in each section.
- View contact details in the Contact section.
