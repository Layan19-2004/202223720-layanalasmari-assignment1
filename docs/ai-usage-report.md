# AI Usage Report

## Tools Used

- **ChatGPT**: Used for code explanations, debugging support, and improving text clarity.
- **VS Code (Live Server Extension)**: Assisted in running and testing the web application locally.

## How AI Was Used

ChatGPT was used to:
- Understand HTML structure and best practices.
- Improve grammar and professionalism in the "About Me" section.
- Explain responsive design concepts such as the viewport meta tag and media queries.
- Suggest font combinations and styling improvements.
- Review and refine CSS structure for better organization.

All AI-generated suggestions were reviewed and adjusted before implementation.

## Benefits of Using AI

- Helped clarify technical concepts quickly.
- Improved writing quality in content sections.
- Provided debugging guidance.
- Suggested design improvements efficiently.

## Challenges and Limitations

- Some AI suggestions required adjustments to match assignment requirements.
- It was necessary to verify that documentation reflected only implemented features.
- Ensuring that the final work reflected personal understanding required careful review.

## Learning Outcomes

Through using AI tools, I learned:
- How CSS inheritance works.
- How responsive design is implemented using media queries.
- How to structure a project using separation of concerns.
- How to review and refine AI-generated content critically.

## Responsible Use and Modifications

AI-generated suggestions were not copied directly without review. 
All code and text were analyzed, modified, and tested to ensure correctness, originality, and alignment with assignment requirements.
