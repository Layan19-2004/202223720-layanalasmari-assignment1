## Project Description
This project is a simple, responsive personal portfolio website built using HTML, CSS, and JavaScript. 
It serves as the foundation for a professional portfolio and demonstrates understanding of frontend development fundamentals.

The website includes:
- An About Me section
- A Projects section
- An Online Courses section
- A Contact section
- Responsive design for desktop and mobile devices

---

## Technologies Used
- HTML5 – Structure of the website
- CSS3 – Styling and responsive layout
- JavaScript (Vanilla JS) – Interactivity
- Google Fonts – Typography
- VS Code – Development environment
- Live Server – Local development testing

---

## How to Run the Project Locally

### Option 1 (Recommended – Live Server)
1. Open the project folder in VS Code.
2. Install the Live Server extension (if not installed).
3. Open `index.html`.
4. Right click and choose **Open with Live Server**.
5. The website will open in your browser and refresh automatically when changes are saved.

### Option 2 (Without Live Server)
1. Locate the `index.html` file.
2. Double-click the file to open it in your browser.
